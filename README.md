# Крипто-бот для Telegram (aiogram + Binance API) + СОВЕТЫ ПО ЗАПУСКУ

Telegram-бот, который показывает цены топ-7 криптовалют с биржи Binance
и даёт простой совет «покупать / подождать» на основе изменения цены за 24 часа.

## Возможности
- Кнопка **Старт** при входе в бота.
- Инлайн-кнопки с 7 монетами: BTC, ETH, BNB, SOL, XRP, ADA, DOGE.
- Цена в USDT и изменение за 24 часа (данные Binance, зеркало `data-api.binance.vision`).
- Простая рекомендация (не финансовый совет).

## 1. Получить токен бота
1. Откройте в Telegram [@BotFather](https://t.me/BotFather).
2. Отправьте команду `/newbot` и следуйте инструкциям.
3. Скопируйте полученный токен (вида `123456:ABC-DEF...`).

## 2. Установка (один раз)
В папке проекта выполните:
```powershell
pip install -r requirements.txt
```

## 3. Указать токен
Создайте в папке проекта файл `token.txt` и вставьте туда **только токен** одной строкой:
```
123456:ABC-DEF...ваш_токен
```
Файл `token.txt` уже добавлен в `.gitignore`, поэтому в git он не попадёт.

## 4. Запуск
В папке проекта (`otc-project`) выполните:
```powershell
python otc.py
```
Бот запустится, если в логе появилось `Run polling for bot ...`. Чтобы остановить — нажмите `Ctrl + C` в этом окне терминала. Окно должно оставаться открытым, пока бот работает.

## Как пользоваться
1. Откройте своего бота в Telegram и отправьте `/start`.
2. Нажмите кнопку **Старт**.
3. Выберите монету — бот покажет цену, изменение за 24ч и рекомендацию.
4. Кнопка **Назад** возвращает к списку монет.

## Логика рекомендации
- Рост за 24ч `>= +3%` — можно присмотреться к покупке.
- Падение за 24ч `<= -3%` — лучше подождать / осторожно.
- Иначе — боковик, нейтрально.

> Это учебный бот. Рекомендации носят информационный характер и **не являются
> финансовой рекомендацией**.
